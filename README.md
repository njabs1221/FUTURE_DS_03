# FUTURE_DS_03 - Marketing Funnel & Conversion Performance Analysis

## 📌 Task Overview
Analyzed marketing funnel data to identify conversion drop-offs,
channel performance, and opportunities to improve lead-to-customer conversion.

## 🛠️ Tools Used
- Python
- Jupyter Notebook
- Pandas
- Matplotlib
- Seaborn

## 📁 Dataset
E-Commerce User Behavior Dataset (Kaggle)
https://www.kaggle.com/datasets/mkechinov/ecommerce-behavior-data-from-multi-category-store

## 📊 Analysis Performed
- Full funnel visualization (View → Cart → Purchase)
- Conversion rates at each funnel stage
- Top 10 categories by views and purchases
- Hourly traffic and purchase patterns
- Monthly purchase trend analysis

## 🔍 Key Findings
- Biggest drop-off occurs between View and Cart stages
- Cart abandonment is a major revenue leak
- Electronics dominates both views and purchases
- Peak activity occurs between 6PM and 9PM
- Purchase volumes show seasonal growth patterns

## ✅ Recommendations
- Use exit-intent popups to convert browsers to cart adders
- Implement cart abandonment email/push notification campaigns
- Focus ad spend on top-performing electronics category
- Schedule promotions during peak evening hours (6PM-9PM)
- Plan inventory around seasonal purchase spikes
